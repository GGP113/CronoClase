Diseño en codigo funcional

Se utiliza Java con Spring Boot para la lógica del sistema, incluyendo la asignación y seguimiento de tareas, así como la autenticación. MySQL se emplea para la gestión de la base de datos, y el front-end se desarrolla con HTML, CSS y JavaScript, conectado al back-end mediante una API REST.

Integración

  • Back-end: java, SpringBoot, MySql

  • conexión Back-End, fornt-End: API Rest

  • Front-end: HTML, CSS, JavaScript
